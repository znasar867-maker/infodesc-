# Bandit OverTheWire – SSH and CIA Triad

This repository documents my hands-on practice with the Bandit challenges from OverTheWire, focusing on secure SSH connections and basic Linux command usage. It also includes my understanding of the CIA Triad, a fundamental concept in cybersecurity.

## Contents

- **banditthewire.md**  
  A step-by-step writeup of Bandit levels 0–5, explaining how SSH was used and how each level was solved.

- **cia-triad.md**  
  A detailed explanation of Confidentiality, Integrity, and Availability, along with examples to demonstrate their importance in information security.
